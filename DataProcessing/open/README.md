# Using the ```open``` File Commnad

### Character Meaning
| Char |                Meaning                            |
|------|---------------------------------------------------|
| 'r'  | open for reading(default)                         |
| 'w'  | open for writing, truncate the file first         |
| 'x'  | create a new file and open it for writing         |
| 'a'  | open for writing, appending to the end, if exists |

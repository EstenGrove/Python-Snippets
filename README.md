# Data Parsing & Processing
NOTE: when working with files on different machines you *need* to be cognizant of the different line-endings that are used on each OS.
__Windows Line-Endings__
- ```CR+LF or \r\n```

__Mac Line-Endings__
- ```LF or \n```
